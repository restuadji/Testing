Materi

Software di client untuk menerjemahkan tag HTML halaman web IE,Chrome, Firefox, Opera, dll

Editor Program untuk menulis tag HTML Notepad, Notepad++, Sublime, Dreamweaver, dll

Membuat tabel : < table > Membuat header : < th > Menambah baris : < tr > Menambah cell : < td >

Di bahasa pemrograman pada umumnya biasanya kita mengenal code, pada HTML kita mengenal tag

Contoh tag:< body >, < head >, < div >, < img > Tag tidak case sensitive. Artinya < body > == < BODY >

Bentuk umum penulisan tag html adalah: < ELEMENT ATTRIBUTE = value>

Dimana:Element - nama tag Attribute - atribut dari tag Value - nilai dari atribut. Contoh: < BODY BGCOLOR=lavender>

CLIENT SIDE PROGRAMMING

Client side programming biasanya lebih banyak dikerjakan oleh front-end developer Komponen: HTML,CSS,JAVASCRIPT

CSS : Cascading Style Sheet Fungsi : mendefenisikan style untuk suatu teks dengan jenis huruf, ukuran , warna tertentu.

Metode Pembuatan : Style Sheet Inline ,Style Sheet Internal, Style Sheet External

STYLE INLINE

mengetikkan langsung dalam tag html sebagai atribut . < body> < b style = “color : blue”> teks tebal dan biru < /b> < /body>

STYLE SHEET INTERNAL

Menggunakan tag style di dalam tag head. < head> < style type=“text/css”> < isi style > < /style> < /head>

STYLE SHEET EXTERNAL

Menyimpan informasi style ke dalam sebuah file dengan ekstensi/type file .css Memanggil file css dalam html dengan tag link yang diletakkan dalam tag head.

< head> < link rel=“stylesheet” type=“text/css” href=“namafile.css”/> < /head>
